# Snake-Game

## About
Snake Game is a classic arcade game developed using Java Swing. The player controls a snake that moves around the screen to collect food and increase its score. As the snake grows longer, the game becomes more challenging. The project demonstrates Object-Oriented Programming (OOP) concepts, GUI development, event handling, sound effects, and game logic in Java.

## Features
-🐍 Snake Movement – Control the snake using arrow keys.
-🍎 Food Collection – The snake grows longer after eating food.
-📈 Score System – Score increases whenever food is collected.
-💀 Collision Detection – The game ends if the snake hits a wall or its own body.

## Technologies Used
Java | Java Swing | Oop | PHP | Java AWT
